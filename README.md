How to edit the content (no coding needed)
The site has a built-in Edit Mode.
Open the live site (or index.html on your computer).
Click the Edit Mode button in the bottom-right corner.
Click directly on any text — names, dates, fees, the director's message, the address — and type over it.
Click any photo placeholder to upload a real photo from your device.
Click Edit Mode again to turn it off.
⚠️ Important: Edit Mode saves changes only in the browser you're using (it does not change the actual files on GitHub). It's best for previewing changes or for the college office to draft text. To make changes permanent and visible to everyone, edit the files directly on GitHub — see below.
How to make permanent changes on GitHub
Open this repository on GitHub.
Click on index.html (or whichever file you need to change).
Click the pencil icon (Edit this file) in the top-right of the file view.
Find the text you want to change and edit it directly.
Scroll down, add a short note describing your change, and click Commit changes.
Wait 1–2 minutes — the live site updates automatically.
No need to touch styles.css or script.js unless you want to change colors/fonts or how things behave.
How the site is hosted (GitHub Pages)
This site runs on GitHub Pages, a free hosting service built into GitHub.
To turn it on (only needed once):
Go to Settings → Pages in this repository.
Under "Branch," select main and folder / (root).
Click Save.
Refresh the page after a minute — your live link will appear at the top in a green box.
To connect a custom domain (like srcpchahdol.in) instead of the default github.io address, add it in the same Settings → Pages screen once you've purchased a domain.
